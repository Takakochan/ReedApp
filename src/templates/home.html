Welcome to The Reed App

This web application is designed for double reed players, reed makers, and researchers who want to systematically collect, analyze, and compare data related to reed cane characteristics.

Whether you're tracking humidity, density, flexibility, or gouging machine details, this tool allows you to select the exact parameters you care about — and store, view, or analyze them over time.

The app includes features like automated density calculation based on dry and wet cane mass, dynamic form generation, and custom data visualizations to support informed reed making and performance decisions.

How to Use the App

Begin by visiting the Settings page, where you can choose which reed cane parameters you want to track — like humidity, density, cane brand, or gouging machine.

As you continue making reeds, enter data for each one using the form. Be sure to assign a unique Reed ID to every piece of cane, so you can update and analyze it over time.

From automated density calculations to personalized tracking, this app helps transform your reed making into a structured, data-driven process.

Built with Django and TailwindCSS, this project merges artisanal knowledge with modern web technology — because great reeds deserve great data.
